Green Cloud Technology

Developed by Marysia Software team Green Cloud Technology is a modular system distributed used for data management. It includes a great number of modules working tightly together. Every single one is an autonomic object, which can work separately, but when working within other modules shows its real power. Users can arrange individually composed systems, according to their current needs. The system has the capability to grow with the user's requirements. New modules and functions can be added without interrupting a working system
